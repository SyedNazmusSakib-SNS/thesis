The Concept: Scene Graphs

Representing medical findings as structured graphs composed of Entities (nodes) and Relations (edges), rather than unstructured text.

Nodule ⎯ located_in → Right Upper Lobe

Key Benefits

Beyond Pixel-to-Text

Moves away from "black box" captioning to explicit semantic representation.

Anatomical Scaffold

Provides a rigid structure that grounds visual features in known anatomy.

Current Limitation

Static Representations

Existing graph approaches (e.g., RadGraph ) are static . They capture a single moment but fail to model how the graph structure evolves or changes over time.
