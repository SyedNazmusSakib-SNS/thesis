Graph Neural Network (GNN)

Methodology | Slide 21

Propagation Mechanism

Change vectors ( Δg ) are propagated through the graph structure using a GNN. This allows localized anatomical changes to influence the context of connected neighbors.

Contextual Outcome

Anatomical Interdependency

Example: If the "Left Lung" node shows significant change, the network automatically updates the context for spatially related nodes like the "Heart" or "Pleura" .

Final Result

H G

Graph Embeddings

Output embeddings that explicitly encode the structural evolution of the disease across time.
