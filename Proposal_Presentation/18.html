Longitudinal Representation Learning

Methodology | Slide 18

The Goal

To explicitly capture "What changed" (content) and "When it changed" (temporal context) between sequential exams.

Core Techniques

Shared Encoder (ViT)

A Vision Transformer extracts visual feature maps from both the prior ( F p ) and current ( F c ) images.

Feature Differencing

Computing the explicit difference vector to highlight interval changes: ΔF = F c − F p

Temporal Cross-Attention

The current image queries the prior image features to identify spatially correlated progressions.

Outcome: Change-Aware Embedding ( Z Δ )

A unified representation focused strictly on disease evolution rather than static anatomy.
