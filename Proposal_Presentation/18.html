Longitudinal Representation Learning

Methodology | Slide 18

The Goal

To explicitly capture "What changed" (content) and "When it changed" (temporal context) between sequential exams.

Core Techniques

Shared Encoder (ViT)

Extracts features from prior ($F_p$) and current ($F_c$) images.

Feature Differencing

Explicit difference vector for interval changes: ΔF = F_c - F_p

Temporal Cross-Attention

Current image queries prior features for spatial correlations.

Outcome: Change-Aware Embedding ($Z_Δ$)

A unified representation focused strictly on disease evolution rather than static anatomy.
